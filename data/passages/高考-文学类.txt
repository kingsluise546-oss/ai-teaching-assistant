# 中高考语文阅读题难度校准表
# 数据基于2021-2025年全国卷及主要省市真题统计

考试  题型          字数范围    题量  建议时间    备注
中考  现代文阅读    950-1350    4-5   15-20min   课外叙事散文/小说为主，偶有说明文
中考  文言文阅读    160-320     3-4   10-15min   课内外对比阅读为主，实词考察为重
高考  论述类阅读    900-1250    3     9-12min    学术/社科论述，逻辑严密
高考  实用类阅读    1050-1450   3     10-12min   多材料组合，信息筛选为主
高考  文学类阅读    1400-1850   3-4   15-20min   小说/散文，意象分析与主旨探究
高考  文言文阅读    520-710     4     15-18min   史传文为主，断句、翻译、文意理解
高考  古代诗歌阅读  40-130      2     8-10min    唐宋诗词为主，赏析简答

# 中考难度特征：
# - 情感主旨明确，人物关系清晰
# - 语言通俗，少用生僻词
# - 题目以信息提取和简单概括为主

# 高考难度特征：
# - 文学类文本有深层意蕴和象征
# - 论述类文本逻辑链完整，有学术化表达
# - 题目要求分析、评价和探究能力
