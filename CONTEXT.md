# AI 教学助手 — 项目上下文

## 项目定位

面向中小学教师的 AI 辅助工具，帮助完成备课、出题、写评语、写通知等日常教学工作。核心交互模式：**点按钮 → 输入一点东西 → 出结果 → 复制走**。

## v1 功能清单

### 页面结构

| 路由 | 页面 | 内容 |
|------|------|------|
| `/workspace` | **首页** | 场景化问候 + 教学提醒 + 最近工作（最近6条） |
| `/assistant` | **核心技能** | 四个技能块展示页（教学分析灰色占位） |
| `/assistant/prepare-lesson` | 备课助手 | 输课题 → 生成教案+提问链 |
| `/assistant/generate-questions` | 出题助手 | 选知识点 → 出三层练习题 |
| `/assistant/write-comment` | 评语助手 | 导入名单+选关键词 → 出N条评语 |
| `/assistant/generate-notice` | 通知模板 | 选场景+填信息 → 生成可复制文字 |
| `/result?id=xxx` | 详情页 | 查看已保存的内容，支持继续优化/收藏/复制/删除 |
| `/assets` | 资产库 | 展示所有已收藏内容，支持取消收藏 |

### 全局功能

- 生成结果自动保存到 localStorage
- 一键复制结果
- 继续优化（基于当前内容迭代，优化结果自动更新 localStorage）
- 收藏 / 取消收藏
- 最近工作读取 localStorage 真实数据，限制 6 条

## 开发约定

- 交互流程：首页 → 核心技能页 → 技能页 → 输入 → 生成 → 继续优化 / 收藏 / 复制
- 数据存储：v1 使用 localStorage，`lib/storage.ts` 统一读写
- UI 风格：Tailwind CSS，蓝白配色，卡片式布局
- 命名规范：页面路由使用英文连字符（kebab-case）
