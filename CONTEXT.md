# AI 教学助手 — 项目上下文

## 项目定位

面向中小学教师的 AI 辅助工具，帮助完成备课、出题、写评语、写通知等日常教学工作。核心交互模式：**点按钮 → 输入一点东西 → 出结果 → 复制走**。

## v1 功能清单

### 页面结构（3 页）

| 路由 | 页面 | 内容 |
|------|------|------|
| `/workspace` | **首页** | 场景化问候 + 教学提醒 + 最近工作（列表） |
| `/assistant` | **核心技能** | 四个技能块展示页 |
| `/assistant/prepare-lesson` | 备课助手 | 输课题 → 生成教案+提问链 |
| `/assistant/generate-questions` | 出题助手 | 选知识点 → 出三层练习题（选择+简答） |
| `/assistant/write-comment` | 评语助手 | 导入名单+选关键词 → 出N条评语 |
| `/assistant/generate-notice` | 通知模板 | 选场景+填信息 → 生成可复制文字 |
| `/result` |（结果页） | 暂保留旧页，后续与技能页合并 |

### 四个技能

- **🧠 教学准备** → 备课助手、出题助手
- **🧠 学生评价** → 评语助手
- **🧠 教学沟通** → 通知模板生成
- **📊 教学分析** → 成绩分析、学情分析（v1占位不可用）

### 全局功能

- 自动保存草稿（localStorage）
- 一键复制结果
- 数据按用户隔离（saved 字段存浏览器）

### 明确 v1 不做

- 资产库（v2 第一个功能）
- 教学分析（留灰色入口）
- 团队协作（留字段，功能不做）
- 预置课标资源包
- AI 自动打标签

## 开发约定

- 交互流程：选择技能 → 跳转独立页面 → 输入 → 结果展示在输入页下方 → 可修改输入重新生成
- 数据存储：v1 使用 localStorage，数据结构预留 userId 和 saved 字段
- UI 风格：Tailwind CSS，蓝白配色，卡片式布局
- 命名规范：页面路由使用英文连字符（kebab-case）
